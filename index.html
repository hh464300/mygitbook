简介

Node下载地址：http://nodejs.cn/download/

Gitbook 官网：www.gitbook.com

GitBook Editor下载地址：https://www.gitbook.com/editor

安装好node后

  • md下输入 node -v 查看版本
  • npm -v 包管理工具
  • 设置淘宝镜像
  • npm config set registry https://registry.npm.taobao.org
  • npm info express 查看是否设置成功

安装gitbook

  • npm install -g gitbook

    再次安装gitbook-cli

  • npm install -g gitbook-cli
  • Gitbook -V 查看版本
  • 正常显示则代表已安装成功

gitbook操作

  • 创建一个目录用于写书 mkdir testBook

  • 必备两个文件 Readme.md 一本书的前言 Summary.md 目录结构

      [简介](README.md)
      [第一章](chapter1/README.md)
      [第一节](chapter1/section1.md)

    启动gitbook

  • 命令： Gitbook serve

  • Gitbook build 输出目录 输入目录 不写默认为_book

#

Gitbook init 初始化 会生成 readme.md和summary.md两个文件
