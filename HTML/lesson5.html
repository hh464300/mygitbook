Css的单位

长度单位：
  1. px 宽高 距离 字体大小等
  2. %
  3. em 字体本来大小的倍数(作用于字体)
  4. rem 根元素html的font-size
  5. vh 1vh = 视口高度/100
  6. vw 1vw = 视口宽度/100
  7. vmin vm和vh中较小的
  8. vmax vm和vh中较大的
  9. 绝对单位：pt（点） cm mm

伪类选择器

  1. :link 链接样式 针对链接
  2. :hover 鼠标滑过的样式
  3. : active 点击时的样式
  4. :visited 访问过后的样式 针对链接
顺序不同 执行效果不同
提示：在 CSS 定义中，a:hover 必须被置于 a:link 和 a:visited 之后，才是有效的。
提示：在 CSS 定义中，a:active 必须被置于 a:hover 之后，才是有效的。
提示：伪类名称对大小写不敏感。
